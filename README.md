# Sistema de Gestión de Disquera

## Descripción

Sistema desarrollado para administrar la información de una disquera, permitiendo gestionar artistas, álbumes, canciones, productores y ventas mediante una base de datos relacional.

## Características

* Gestión de artistas.
* Gestión de álbumes.
* Registro de canciones.
* Administración de productores.
* Control de ventas.
* Gestión de usuarios.
* Consultas y reportes sobre la información almacenada.

## Modelo de Datos

El sistema contempla entidades como:

* Sellos Discográficos
* Artistas
* Productores
* Álbumes
* Canciones
* Colaboraciones
* Ventas
* Usuarios

## Tecnologías Utilizadas

* C#
* SQL Server
* Windows Forms

## Objetivos del Proyecto

* Aplicar modelado relacional avanzado.
* Implementar operaciones CRUD completas.
* Gestionar información musical mediante una base de datos estructurada.
